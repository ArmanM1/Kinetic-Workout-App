## 2026-04-28 - iOS Native Search Keyboard for Mobile Search Inputs\n**Learning:** Using `type="search"` on an `Input` element triggers the native 'Search' keyboard button on mobile (especially iOS), improving the UX significantly over a standard 'Return' key. It's also important to add an `aria-label` for screen readers since these inputs only have a visual icon and a placeholder.\n**Action:** Always add `type="search"` and `aria-label` (or a visible label) to utility search inputs.
